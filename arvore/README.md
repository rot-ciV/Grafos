# Árvores Geradoras Mínimas (MST)

Todo grafo conexo possui uma subárvore que contém todos os seus vértices. Uma Árvore Geradora de um grafo é um subgrafo abrangente, conexo e acíclico. Quando as arestas possuem pesos, o objetivo passa a ser encontrar a **Árvore Geradora Mínima** (Minimum Spanning Tree - MST), que é aquela cuja soma dos pesos das arestas é a menor possível.

Nesta pasta, estão implementados três algoritmos para busca de Árvores Geradoras Mínimas.
